因信称义，因信而活 #

《加1:6-9；2:16,20；3:26-29；5:1,16-26》

序论：一个人信什么，他将会成为什么。倘若一个人听信了全备的基督福音，他将回复上帝创造他原有的样子，充满恩典与真理，管理治理一切，也生养众多（祝福万民）。但他若信了非福音（或部分福音）的教义，从那里发展，他里头的心灵将会受到不正直与不清洁的灵影响，导致他行事为人仍存留在《创3》的堕落光景。加拉太书（与罗马书）是新约圣经里将福音的精义讲解得最清楚的书。这本书的背景讲说在加拉太（小亚细亚的地区），保罗初次将福音传到这般外邦人当中（使13-14章：彼西底安提阿、以哥念、路司得、特庇），他们听见就被圣灵感动，欢喜接受那使人得自由的福音，圣灵因此也在他们当中大大的做工，赐他们自由喜乐与异能在他们中间。但过后不久从犹太人中来了一群不认识福音的热心份子。他们高举犹太人的礼仪、节期、形式，强调这些在基督的救恩里是不能缺少的条件，不然人在神里的称义无效。保罗知道这完全是邪灵的工作，是根本推翻唯独基督信仰与消灭圣灵工作的教导。当加拉太人得着这些教义为信念时，他们的心灵被辖制，失去福音所赐的自由，专追求守犹太人的律法（行割礼与其余规条），甚至好些也对初次向他们传福音的保罗起了意见。这都是因为他们被“基督还不够”的信念蒙骗了！从这里，我们会晓得，若一个信徒在因信称义的绝对真理上被摇动了，他心灵的光景便会全然进到奴仆的状态，他在整个信仰生活当中也无法享受顺着圣灵（儿子的灵）的感动而活。在这等光景，就算他达成了一切律法的准则，他的心灵也无法满足神的标准，因为所想所行的都是因惧怕或亏欠神出发，不是因蒙恩蒙爱蒙召的心灵出发。这也是我们今日信徒要确实解决的弊病，好叫我们得因信称义的确据与自由，以致行事为人都是因信（寻求与随从圣灵的吩咐）而出发。据说，马丁路德在改革宗教时常引用加拉太书里有关“因信称义”的真理《2：16》，来推翻罗马天主教当日弯曲的神学与救恩理论。所以，让我们接下来透过《加拉太书》多认识“因信称义，因信而活”的关键内容。

福音是不能加减的！《1：6-9；3：1-5；4：8-11；5：2-4》

福音的中心就是：「唯独基督称义」 #

福音的最关键就在于基督的唯独性。人成为蒙恩、蒙爱、蒙福、蒙召是单单透过基督的十字架，不在于人义。唯独听信基督，人得永生，罪孽得赦免，得儿女的名分，领受所应许的圣灵。因此，圣灵在一个圣徒心中的做工（印记、感动、充满、恩膏、结果子），都是借着他单单认识、寻求、信靠基督，并非人的虔诚或善行而来。若是加减了，便会拿走福音的功效（神的大能）。

所以，信徒里头最危险的意念就是：「单单有了基督还不够」 #

亚当夏娃的堕落来之被蛇欺骗，认为有了神的形象(话语与圣灵)还不够，今天信徒最大的失败就是一直认为有了基督还不够。换句话：不信在基督里已经拥有完整的祝福与成功，反而认为除了拥有基督以外，还需要而外的祝福、成功、漂亮、聪明、条件、等。从这里，人的生命马上会受试探，在上帝所赐完整的应许与自由（基督），加上各种人的条件、准则、律法、等。

若拥有了基督还要加减任何人的条件、准则、律法才完整，圣灵的工作便马上削弱，欺骗者甚至会逐渐败坏人起初的信仰 #

欺骗者是特别引用上帝全备的律法来控告神所爱的人，使他们认为自己因没有达到圣经某些律法的要求，而感到自责亏欠。但圣灵的工作是叫人看见自己的不完整而扎心，因此愿意亲近父神，从祂得能力行善。这两者的背后是不同的灵带领。所以，当一个人继续以守律法与某些条规为信仰准则，他很快会失去与神的父子联系，里头逐渐产生是非之心，以对错为良心标准，不是以爱神爱人为标准。这将导致他的良知时常受控告（因自己无法完整），但也会促使他在行善之后自以为义、论断人、要求人。

律法使人知罪，福音带来神儿女的确据（单信基督而称义）《2：15=21；3：6-14；4：1-7》 #

律法不但指圣经中的字句条规，更明确说是人的是非之心。当不认识神的爱的人领受了有些教义后，便按是非之心判断一切。 律法的功用是叫人知罪，使人知道没有任何靠人出发的善意善行能达到神的标准 所以，人不知罪或自己认为自己能解决罪就是最大的罪了！神律法的严谨与全备，就是要堕落的人知道人的里外都无法达到律法的要求。但若有任何一个宗教或教义使人认为能使人脱离罪与罪的刑罚，那将受到上帝最大的诅咒。另外，当人不知罪（不感到扎心），必不来到赐宏恩的主面前，他便会造出更多的律法来安逸自己受控告的良知，以此为自己的义。

知罪（无能）之人，才能领会福音所赐神儿女的生命 #

上帝所预定为儿女的人，已经在创立世界以前就爱他们了。但若还未知罪（无能），他绝不能领会上帝的恩典，也无法领受儿女的心（圣灵）。所以，神将他生在这世界，为奴的光景，直到他领悟了世界（肉体）的空虚与自己的无能，神的福音才向他显明，使他领受在基督里神儿女的确据。这确据就是他心里所得的圣灵的印记，叫他肯定他在基督里是完全蒙恩、蒙爱、蒙福的。

唯有回复儿女的生命，才能明白全律法的精义就是“爱神爱人” #

当一个人回复神儿女的心重看上帝的律法时，律法已经不是辖制他，定他罪的宪法了，乃是上帝的心肠显明在里头。这心肠也就是我们主耶稣基督所显明的“爱神爱人之心”。上帝一切的美意都包含在“爱”的心肠里。事实上，福音的精义就是爱，在爱里没有惧怕，也没有对错之分，乃是为了造就生命，使主的身体合而为一。若一个信徒有爱神爱人之心，他虽有时在所行的事上不太合宜，但无限的圣灵将会指教他，也能纠正他，那也正是神所要的“听从神”的信仰。

福音叫人不再受律法的要求辖制（得自由），乃顺着圣灵而行《5：1，13-26》 福音的结论是叫人顺着圣灵而行，就是在凡事上以“爱神爱人之心”寻求何为神的“美意”而遵行 已经借福音得以自由的信徒只需透过心灵肯定两件事：（1）人生的目标定为荣神益人建国；（2）人生的方法只有“听从神为主”。当一个人带着这两种心灵过地上的人生，圣灵必在凡事上无限的向他显明，使他一生出也蒙福，入也蒙福。今天，许多信徒虽是去了教会，还是不能在心灵里肯定这两件事，所以劳苦重担，反倒在许多事上钻牛角尖，其实都是不顺从圣灵而来的。人为何在行事为人给自己很多要求，加上许多准则和戒律？其实都是因为爱自己多过爱神，因此不愿意放弃自己的意思而遵行主。

事实上，当人明白如何顺着圣灵而行，才不放纵肉体的情欲 #

很多人专强调说：“你们不要犯罪，不然圣灵就离开，或就是圣灵不再感动你！”其实，这等教导是片面性的。因为当人唯有晓得如何顺着圣灵行，才能不放纵肉体的情欲。所以，我们要将人从罪意带出来，先要将“神如何爱他”的事实告诉他，再教导他如何察验神的美意，与确认神如何祝福他的证据。这样一来，人才能从罪意里渐渐得自由，归向神。简单来说：当信徒多听见神的话（应许），才不受世界的意念干扰；当信徒多以肢体、以祭坛、以事奉为中心生活，才不会一直留在软弱里。

顺着圣灵行之人必有圣灵的果子在他身上《加5：22-23》 #

纵然，我们总要谨慎，因为许多人不明白圣灵所赐的自由，他污秽的良心常误认自己所行的是出于圣灵。所以，保罗特别强调说：“那顺着圣灵之人必在生命里会结出圣灵的果子”。这内在与显露出来的果子是我们在凡事里常要确认的。若我们明白与按着圣灵的指示而行时，里头的圣灵必给我们这些果子为证据，在我们身上也逐渐显出这类果子，使我们知道自己是行在圣灵里。

福音最终强调的是新造的人，不是仪文的旧人《加6：15-18》 #

当一个人常顺着圣灵行，最大的祝福就是他里外越来越回复基督的荣形，爱神更深，爱人越真实，对凡事上主的美意越清楚。他心中的意念都为神所喜悦，祷告就看见应允，顺服就得见赏赐。这就是信徒所当享受在基督里的自由人生了。
