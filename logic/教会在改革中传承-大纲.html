《罗11:36》 #

序论：可能有些人看了今天的主题会感到疑惑，因为圣经明明说教会是建造在使徒和先知的根基上《弗2:20》，为什么说是建立在改革的信仰呢？改革的教义又是什么呢？那跟圣经又有什么关系呢？首先，我们要清楚知道主耶稣说：我要把我的教会建造在这磐石上《太16:18》。很明显的，这里所说的磐石不是指彼得这人，乃是彼得这人所信所传的那位房角石基督。确实，我们所信的是基督，但谁将基督和有关祂的一切启示带出来呢？那就是神所委派写新旧约的【使徒和先知】了。使徒是基督所拣选，且是亲眼见过主，之后从圣灵直接领受启示的人。在使徒的时代过了之后，将使徒的书信教导与传递的人就是【教父】。他们虽没有亲眼见过主，但却是受使徒们教导，且受使徒们亲自委任来带领教会的人。教父中有几位非常著名的，如：罗马的革利免，安提亚的伊格内修斯，或是士毎拿的波利卡等。在使徒与教父时代，我们从使徒的书信知道错谬的教导已经开始盛行。使徒保罗也透过圣灵的启示明说：在后来的时候，必有人离弃真道，听从那引诱人的邪灵和鬼魔的道理《提前4:1》。因此，教会是继续借着使徒们各自的书信来被提醒，且被带到正确的教义。一直到了主后第四世纪的几个会议中，圣经的正典（66卷）被保守派的基督徒与福音派教会承认为独一无二的神的默示。就在靠近当时的世代，神也赐给教会一个非常重要的人物，就是【圣奥古斯丁】。他不但在定出教会正典扮演了很重要的角色，也是第一位将圣经神学教义很透彻地领悟的神学家。因此，他是在教会界被公认为神学之父，也可以说在教会历史的第一千年最伟大的神学家。无论是天主教或基督教，他都被公认为给予教会最大贡献的人物。有的人也许会问为什么教会在奥古斯丁之前似乎都没有正式的神学教义呢？那是因为当时圣经的正典还未被定准，因此人对神心意的理解还是留在局部性的。

但是当圣经的66卷书被定准之后，圣徒对神的心意已经拥有一个完整的概念。像奥古斯丁这样的人物出现，将圣经里的神学教义参透与结合在一起，以致叫信徒能清楚地认识神与祂的救恩，且不偏向错谬的教义，这本是神对教会的本心意。当然，教会历史里充满波折。从圣经正典被定准之后，仇敌不停止工作。因此，人的罪促使他渐渐将正典的启示扭曲，因此教会就进入了所谓黑暗中世世纪。这世纪长达了差不多一千年。当然，在这一千年，神仍然保存真理的见证人。但整体来说，我们渐渐看到政教合一，基督的恩典与救恩被扭曲，因信称义的教义被削弱，到一个地步教皇享有了几乎绝对的权柄，教权被提升为最高的权威，信徒也可用赎罪券来赎买自己因犯罪当受的惩罚。就在这腐败持续地盛行，在1517年，神使用一个修道士马丁路德写了《九十五条论纲》，钉在教堂门上，主要是邀请人来辩论有关罗马天主教现行的教导与习俗，谁知神大能的恩手借此卷起了基督教历史最伟大的宗教改革运动。在1521年，马丁路德被赶出罗马天主教，但这由神促使的改革运动也获得胜利，教会就从那时候起脱离了那已经衰残与腐败的天主教。因此，从改革到现今已经有差不多500年。当时，有许多著名的改革家都兴起。其中一位与马丁路德齐名的神学改革家【加尔文】也在当时出现。神透过他将极全备又牢固的教义带给教会，而他其中一个最经典的作品就是基督教要义。这基督教要义是基督教经典中的经典，不但坚定了正规的神学体系，也大大的塑造了接下来的教会历史，甚至它也深刻影响了世界历史。无论是在政治或教会治理的指南上，它都给予很重要的根基与原则。

有一个神学家说：在过去五百年的西方世界里，每当社会出现危机、根基开始动摇、人心感到恐慌的时候，人们就会带着崇敬之心，重新研读这部经典。因此，从这些主兴起的人物带来了基督教公认的“改革信仰”与背后牢固的“改革教义”。那么，我们问这改革信仰只是长达500年吗？绝非如此。当我们说宗教改革500年周年纪念，那是从马丁路德将《九十五条论纲》钉在教堂的大门上算起，但改革信仰里头的教义并非只限于500年。其实，那是纯粹忠于神借使徒与先知所带来的教导。倘若我们细读马丁路德与加尔文的作品时，我们也会发现他们不是历史中间出现的独行改革家。他们乃是重视神一来在众教会里的工作，他们的作品中最多引用的是圣奥古斯丁所留下来的教义，而奥古斯丁所领悟出来的教义是建立在教父所传，且是使徒们从圣灵所领受的默示。当然到如今21世纪有很多伟大的神学家与牧师出现。但是我们将这些刚刚所提的人物列为那带来圣经正道的主干人物。因为即便当基督教广传时，有许多不同的宗派，甚至在某些圣经的教导与原则上有不同的强调，但最终能存留下来，且继续大而有力地影响每个时代的还是这些主干人物留下来的教义。有些人说：亚米念思想也影响了现今众教会很多福音派。但我们有否问过：为什么神不用信服亚米念思想的群人来兴起宗教改革，反倒用那拥有改革信仰的人呢？为何亚米念思想对神的主权与恩典的认识与奥古斯丁的大不相同呢？有时看起来现今讲恩典主义的牧师和教师吸引成千上万的人进入他们的教会，但为何他们所讲的恩典与加尔文、马丁路德、奥古斯丁讲的恩典不同呢？为何他们不像这些正道的主干人物一样传讲人的罪与神的震怒呢？从这里，我们就会知道这些恩典主义的牧师与教师是在选择接受那符合自己和自己群众的教义。曾经在线上有一位从传统到灵恩教会的弟兄挑战我说：“看看现今五旬与灵恩教会的扩展！看看他们教会的人数！那些基要派、保守派，甚至传讲改革教义的教会都不能带来这样的复兴与震撼力！最关键的是大多数的五旬派与灵恩派都不接受改革教义”。其实，这样的看法与观点是又肤浅又短视的。首先，我们要问：“难道多人进入的宗派就是复兴吗？”复兴不是先从外在看起，乃是从本质发觉。在一个大群体里，我们要问多少人是真正借着神的真道而重生？当然，我们看不到人内心的光景，但是从一间教会所传扬的讲台内容，我们会晓得下面的人信的是什么。在没有将神的福音讲解清楚的教会里，我们会看见一个普遍现象，就是看起来信的人多，最终跌倒的人也一样多。

另外，当我们因人数的增多而叹为观止，我们一定要问是哪一种教义影响世世代代的国家、社会、家庭等？当我们看见整个西方界的政治体系、法律体系、经济体系，连教育体系都带有基督教痕迹时，我们就晓得这些背后都是改革的思想带来的。这绝对不是搞一下盛大的活动与盛会，邀请政治人物来参加，或相似给我们剪彩这类这么表面的事。并且，若我们以全球的数字来看，福音与保守派的信徒还是占百分之七十五。五旬和灵恩的信徒只是占百分之二十五。但就因为在五旬与灵恩教会有少数非常聪明与杰出的领袖，懂得将人吸引进入教会，我们就很片面地认为他们正在经历大复兴，其实相反的，他们是一直在拼命地维持不让自己走到没落。最后，历史也告诉我们，真实的道理是绝对经得起时间与患难的考验。它不需要伪装自己，或是继续推陈出新来吸引人。改革的根基与教义从奥古斯丁、马丁路德、加尔文到至今都没有改变，但是在很多离开改革教义的福音派，以及不接纳改革教导的五旬与灵恩派，我们看见他们继续要使用新鲜的方式，以及各种推陈出新的信息与方法来将人吸引到教会。这就如社会福音、医治释放运动、信心运动、恩赐运动、恩典福音、新使徒运动等。其实，这些以人为本所发明出来的教导与燃起的运动根本是没有中心点，像是快要熄灭的灯火。这是因为他们的起首并不是要忠于真理，且将自己一切的思想、信念、教义、生活、家庭、教会等都附属在圣经全备的真理之下。一旦他们察觉那是不能取得自己想得的利益的教导，他们就舍去那样的教导与原则，接受另一种比较能够配合自己的经历与需要的教义。这也就是当代教会现今所面对最大的试探。这也就是为什么我们需要回归与持守正道的理由。

1. 为何叫改革（或：归正）信仰？ #

1）改革（或归正）的大前提 #

当我们说到改革或者归正，那就是说现有的已经是彻底腐败与错谬了，像已经腐败的北以色列，或是已经完全失去神心意的犹太教，或者就是已经宗教化的天主教，因此才需要改革，需要归正。而且，我们需要改到什么地步呢？是表面改善吗？绝不是！乃是在根基上需要改。因此要改革，需要“全部拆毁，重新建造”。因为在根基上已经错了。当我们看到外表的形式、作风、生活已经错了，也不断出现错谬的形态，我们知道问题在根基。因此，当我们将腐烂的根除去时，才能开始生根建造，信心坚固。

2）那么，改革是谁带来的？ #

宗教改革看起来是马丁路德带动的，但其实我们从教会历史都清楚知道这改革是神主动的。祂感动祂的仆人将《九十五条论纲》钉在德国维腾堡教堂的大门上。这一事件，在当时非常微不足道。路德本人只是出于义愤和牧者的心肠，寻求一场公开的辩论，以澄清滥用赎罪券的问题。在当时的学术界，这是司空见惯的事情。然而，因着上帝主权性而奥秘的护理之手，这一微不足道的事件却最终成为一场轰轰烈烈的“宗教改革运动”的导火索。为什么神要许可这样的事呢？因为神的名，神的荣耀受到威胁了。因此，我们晓得那腐烂的根不是别的，就是教会对神的认识已经完全错误了。因为一切本于祂，倚靠祂，归于祂《罗11:36》。当我们对“祂”错解与错认时，神的荣耀就被遮盖了。因此，改革教义的归正就是从归正我们对神的认识开始，而这认识绝不是凭着我们的经历或体验，乃是纯粹透过神的道和我们忠于基督的信心而来。 因此，我们在信仰里一切的问题是源自对神错误的认识。

3）归正我们对神的认识 #

在《弗1:17》，我们看见保罗为以弗所教会信徒的第一个祷告就是求神将那赐人智慧和启示的灵赏给他们，使他们【真知道他】。不是多知道一点，不是局部性的知道，或是理性里面的知道而已，乃是真知道祂。因为从我们对神的认识，会断定我们对万有的认识。特别在改革里面强调我们对神三方面的认识：① 神的圣洁。神的圣洁是祂与万物的分别性。祂是王，但祂不与世界的王一样。祂是神，但祂不与世人拜的神一样。祂是爱，但祂的爱是超越世人的爱，也截然与世人的博爱是不同的。祂的爱圣洁到一个地步，以致祂全然的爱专属祂的子民（换句话：祂的新妇）。② 神的主权。因为祂是全然圣洁的，因此祂拥有绝对的主权。祂要怜悯谁就怜悯谁，要恩待谁就恩待谁《罗9:15》。不但如此，万有里一切所发生的事都在祂的命定之下发生，祂决定一切事情的发生与发展。但祂也是那位使万事都互相效力，叫爱神的人得益处，就是按祂旨意被召的人《罗8:28》。因此，即便罪与邪恶在祂的命定之下临到世界，但救赎与恩典也同时临到祂在基督里预定所拣选的人。因为一切本于祂，倚靠祂，归于祂，愿荣耀归给祂，直到永远！阿门《罗11:36》。③ 神的真理。在神圣洁的荣耀与主权性的恩典里，祂将祂的真理启示给人，使人能够借此认识祂和一切在祂里头的知识。因此，整个基督徒的信仰生活是从我们得着真理的启示开始的。基督教与所有宗教最大的差异就是我们是领受了神真理启示的人。但所有宗教都是在人的层面，借着一般启示来领悟出一些道理。因此，当我们面对地上一切的课题时，我们是连接于我们对神和神里头的知识。例如：当我们谈到婚姻，我们是从“神所创造的婚姻”开始，不是“相亲相爱的婚姻”开始。当我们讲到事业，是从“荣耀与服侍神”的事业开始，不是“我要成功或是找到让我感到有价值”的工作开始。无论我们讲到什么课题，当我们将“神”从那课题拿掉来采用什么圣经原则，那已经不是神的真理了。那只不过是人利用圣经来制造出来的人理。但从另一方面，我们也要小心的就是在改革教义的丰富知识里，我们不要落入“知识主义”。因此，改革的精神不是我们采纳一些学术性的观点叫做“主权性的恩典”或是什么“加尔文思想”，乃是叫我们全然的忠于圣经。如之前所说，真正的改革是将自己一切的思想、信念、教义、生活、家庭、教会等都附属在圣经全备的真理之下。

2.学习与教导改革教义的挑战 #

1）要谨慎不落入知识与戒律主义 #

有很多传讲改革教义的牧师落入一种没有生命力的知识。很多学习改革教义的信徒也只学到知识，但并没领会改革信仰的精神。有些改革宗教派看见福音与灵恩派的教导越来越偏向实用主义，就偏向非常知识性的极端，认为在理性上认识那牢固的神学教义就够了。其实，真正的改革信仰是深信神向人所启示的道是人能理解的。那是不限制于人的学问与文化水平的。只要是愿意借着真理来寻求神的人，无论是男女老少都能理解。因此，他即便有丰富的知识，但绝不是一种高超与深不可测的哲理。我们一定要记得保罗为以弗所教会有两方面的祷告。第一是求神使信徒真知道祂《弗1:17》，再就是求神使他们心里的力量刚强起来，且因着信而享有一个有根基的爱心，甚至那等心是能与众圣徒联合而在一起发现基督无限的大爱《弗3:16-18》。因此，我们“真知道神”的果子是实现在我们心中和我们与众圣徒之间的关系里。因此，如果改革教义带来分裂与分歧，或只是一种纸上谈兵的理论，那我们就会使改革教义受到很大的亏损。即便改革教义里有浓厚的护教因素，但护教不是为了要证明自己对，乃是为了要牧养主的群羊。

2）信仰的根基要与生活的应用联结在一起 #

在引用改革的教义来牧养与劝诫人时，我们往往需要考虑到根基性的问题与应用性的问题。所谓根基性的问题是人信念的问题。应用性的问题往往是限于人眼前能够看到与察觉的问题。例如：当我们问为什么现代信徒常说自己不能享受与神交通和同行呢？一般信徒都是从应用性的角度来看待这个问题，因此就会说：“因为手机的试探，或是因活在资讯爆炸的时代，这些常常将我门的注意力带离神，或者就是有什么问题就问google，不用问上帝，要不然就是城市生活很忙碌，以致没时间与神亲近”。这些是实际性的问题。但是改革信仰却是装备我们以一个很根基性的观点来看问题，就是我们不能与神交通和同行是因为我们没有对神有一个清楚的认识，对真理一知半解，因此没有办法活在祂面前。我们似乎是常常学习，终久不能明白真道《提后3:7》，因此有敬虔的外貌，却没有敬虔的实意《提后3:5》。在此，圣经将人很深入的信念问题带出来。但是圣经也说到这等人身上也实际的有19种不敬虔的特性《提后3:2-4》。 因此，改革信仰教导我们怎么看到问题的根源，但同时也能够认定实际性的问题。由此可见，真正的改革信仰不是一直专讲神学、历史、教义，却没有应用性的，因为神造我们人的生命是有肉身的。因此，神赐给我们的话语是继续要经过确认、察验而顺服，才能实践的。但是在所有的应用里，我们不能只留在专分析现实的问题来让人留在贴切感而已。我们最终是要将人带回到对神的认识。这样一来，我们所传讲的就是那使人在基督里生根建造，信心坚固的教训了《西2:7》。

3）求主赐下恩惠，使我们做一个无愧，能按着正意分解真理的道的信徒 #

当神将这么重要的改革教义赐给教会时，我们真的当求神使我们能在祂面前蒙恩。因为即便有这么好的教导出现，但人在应用时也会偏向自己的倾向。有些人偏知识，就走向知识主义。有的人偏应用，就走向实用主义。久而久之，那个偏向就会结出果子，在人的生活与关系里显露出来。确实的说：我带着时代性的使命，要将主的道传讲得全备，好服侍众教会，但主也将一个家庭里作丈夫与父亲的责任赐给我。我怎么对待我周围的人是必须合乎我传讲的正道。倘若我没有爱护我的妻子，且不管孩子的成长，我的家人也不会信服我所传的正道。那这是不是他们只在乎人的爱，不在乎神的真理呢？其实，当我这么认为时，就是无形中将道与生命分开，神的话与圣灵分开，或是心灵与肉身分开了。那么，实用主义就会乘机卷起。主耶稣说：“我就是道路、真理、生命”《约14:6》。祂不是多一点道路与真理，少一点生命，也非反之亦然。因此，改革信仰的正道终久是叫我们活出基督
