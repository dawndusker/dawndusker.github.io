《民6:22-27》《太6:1-6；16-18》 #

序论：基督徒信仰生活的总概念就是“活在神的面前”。这也是在宗教改革之后，改革家呈现给世人的基督信仰大概念。确实，不管我们的处事待人多么的良好，我们的服侍多么的有果效，最终我们所得的赏赐是借着我们行在主面前而来的。其实，当我们的良知是安置在神的话语上时（换句话：我们对是与非的判断是以神的道为基准时），接下来信仰生活的一切功课就是活在神面前的事了。有些人的祷告是在神面前的祷告，他的祷告是得父的赏赐的祷告，但有的人的祷告却是自言自语的祷告。有的人的服侍是行在神面前的服侍，他的服侍是得父称许的服侍，因此力上加力，但也有的却是行在人面前的服侍。但我们也当晓得有时候，可能有些人的生活服侍是离开神的面来做，他所做的看起来反而越做越好，越得人的认同，但那绝不是兴盛，乃是败亡。因为他的下场就会像扫罗王一样，越做越骄傲，越服侍越靠自己，然后一旦出现了比他更能干的人，就开始怀妒忌与惧怕的心。即便他得了很多人的爱戴与肯定，他无法感到稳妥，因此最后以一败涂地来收场。但是大卫刚好与他相反。对大卫来说，即便他周围遍满了人与事物，他被各种的环境围绕，有时顺境、有时逆境，但他最渴望的，就是一生一世住在耶和华的殿中，瞻仰祂的荣美，在祂的殿里求问《诗27:4》。其实，我们所有的装备不是最终要活在神的面前吗？我们判断是与非不就是要在一切的事上行在神面前吗？但往往，我们很难以人暂时外在的情况来判断他是否活在神面前。因为外在所发生的事只是最终的结果，但我们不完全晓得一个人的内心与主之间的事情。扫罗40年坐在以色列的王位，但大卫许多年飘无居所，直到他成为全以色列王的日子。确实，我们人的心是很诡诈的。有些人常祷告说：“主啊，我们在祢面前是怎样行、怎样活...”，但他之后所想、所讲、所做的却是与神的圣洁和公义相反的。这里毁谤人，那里重伤人，不然就是贪图虚名，要得人的肯定，或者就是将自己的生命浪费在宴乐中。但问题是他不是不知道，他乃继续放任自己活在这样的光景里。渐渐的，他心中就开始对神的道没有感觉，且听不进神借祂仆人所带来的教训、督责、劝勉等。那也就是他渐渐离开神的面了。其实，一个真正的基督徒知道没有比离开神的面更危机的事。当我们不能因所知道的神的道而来到神面前，那是我们要警觉的时刻了。当大卫因他的罪悔改时，他说了一句非常重要的话。他说：“神啊，不要丢弃我，使我离开你的面，不要从我收回你的圣灵”。对他来说，没有比离开神的面更可怕，因为他就是离开神的面而陷入一次又一次的犯罪，甚至对自己所做的没知觉。当一个人因自己的私欲开始对神的正道产生抵触时，他已经是开始离开神的面了。他之后会因自己所要的，继续走妥协的路。渐渐的，他的良心就会丧尽。这样的人就如保罗所说的：有人丢弃良心，就在真道上如同船破坏了一般《提前1:19》。为何说他们丢弃良心呢？因为他的良心确实借着他起初认识的正道来约束他，但渐渐的，他因着自己的私欲（自己要得的）淹没那良心的指责，去信从另外一个道了，到最后连良心的警觉都听不见了。保罗说要将这等人交给撒旦（换句话，让他们自己走世界的路，遭受应得的结果）。其实，若不是神的保守，我们每个人都有可能走到那条路，丢弃那纯正的教义，选择随从那合乎我们心意的道。因此，既然我们晓得了仇敌的工作，我们更是要谨慎自守，常存信心和无亏的良心活在神面前。其实，活在神的面前不是一朝一夕的事，因为我们与生俱来有一个看不见神的本性。因此，我们不是看见主，反而总是先看环境。我们不是先依靠主，而是总先依靠人。以色列民就是无法看见神的面，因此渐渐的，他们开始要求一个君王来统治他们。同样的，若一间教会不懂得寻求神的面，整个的运作就会开始世俗化，人的话就多起来，神的话就稀少起来。这都是很微妙的会发生的。但我们当晓得活在神面前也不是我们自己去想象出来的事，乃是按着我们所信的道活在神面前。因此，正道是多么的重要。因为人诡诈的心能够叫他随意信道，且随意寻求神的面。圣洁的神绝对不会向这样的人显现。因此，我们要从今天的信息得知怎么真正活在神面前。

  1. 谁能够来到神面前？ #

1）唯有神自己先向我们仰脸，我们才能得见他 #

在旧约中，神嘱咐亚伦大祭司给以色列民的祝福有三句话，但意思是一样的。神赐福给我们，保护我们；神使祂的脸光照我们，赐恩给我们；神向我们仰脸，赐我们平安；这都是一样的意思。倘若神掩面不看我们，就是等于祂不祝福我们，也不赐给我们平安。确实，没有比神向我们仰脸更大的祝福。有时，我们在崇拜完毕时，牧师代替神给予会众这祝福。这并非是一种形式性的祝福。这个祝福告诉我们三样事实：① 神的本意是愿意祝福祂的子民的，祂也愿意祂百姓的领袖为祂百姓在神面前求这样的祝福；② 神要祂的百姓晓得一切的祝福是来自神，若不是神祝福，没有一样祝福能临到祂百姓身上；③ 神要祂的百姓晓得祂的祝福不是来自别的（丰收、战胜仇敌、事事顺利等），乃是来自神向他们仰脸。因此，活在神面前的人才能看见神的平安与保护。

2）唯有借着信才能得见神的面 #

今天，谁才能看见神向他仰脸呢？我们要确实的晓得，我们不是借着善行而得拯救的。因此，不管我们的行为多好、功德多善、成就多大，没有一个人能依靠这些来到神的面前。我们能够看到神向我们仰脸，也能够期待祂祝福我们，是因为父神借祂儿子耶稣基督向我们显明的恩惠。基督的顺服就是我们的顺服，基督的功德就是我们的功德。这一切都是借着我们在神儿子里头的信心而来的。因为经上所记：“义人必因信得生”《罗1:17》。确实，真的能活在神面前的人是那唯独以信心站在神面前的人。他可能是一名很有爱心的牧师，是很忠心的事工者，是很慈祥的父母，是很尽责的员工等，但他是因信活在神面前，也因信行出神的旨意。圣经说信的人在神面前是蒙祂称许的，因为他们在信上得了美好的证据《来11:39》。其实，很多时候，我们看不见神的面，看不见神向我们仰脸，为何呢？因为我们还是有所依靠自己的义、自己的条件、自己的功德、自己在人面前的地位等。因此，我们一旦受挫、受屈、受苦、受伤等，我们不是以此靠近神，乃是远离神，那之后的反应就显露出我们所信的是什么。因此，一个真正活在神面前的基督徒是凭着他所信的（基督），不是凭着他所行的（功德）。

3）但谁是那真信的？ #

不是宣称自己信的人，乃是那【遵行神旨意的人】。心中真信主的人是凡事都愿意做在主面前的人。主耶稣说：凡遵行神旨意的人，就是我的弟兄、姐妹和母亲了《可3:35》。其实，整个信仰生活可以说是活在神的面前，但用另外一句话来说就是遵行神的旨意。因为那真信主的，在心中同时渴望遵行神的旨意。《弗6:6-8》清楚说：不要只在眼前侍奉，像是讨人喜欢的，要像基督的仆人，从心里遵行神的旨意，甘心侍奉，好像服侍主，不像服侍人。因为晓得各人所行的善事，不论是为奴的，是自主的，都必按所行的得主的赏赐。因此，真信的人不单单感觉自己有信心而已，他一定有信心的明证。信心是将他带到神的面前，叫他能凡事求问主、依靠主、好遵行主道。今天，很多人误以为自己很迫切地向主祷告，就以为他是来到神面前。但是当我们仔细的听他的祷告，他并非是要遵行主的旨意，他乃是要神来成就他的意思，或是求神来帮他做事。这就是真信和假信的差异了。有些人很矛盾地说：若我遵行神旨意，就要放弃我的梦想了，不能做我要做的事。这是根本自相矛盾的信仰，是一个信神却不喜悦神的信仰。因为他将神的旨意当做是禁止他的，不是祝福他的。试问这样的人怎么有蒙福的确据呢？他怎么能看见神向他仰脸呢？相反的，他是逃避神的面的人。这样的人不会从神得着什么。

  1. 活在神面前，得着神的赏赐 #

1）主说：“你们要小心” #

为何要小心呢？因为一瞬间会落入“不面对神，且不活在神的面前”的光景。特别在《太6:1-6;16-18》，这里提到三样犹太人最重视的个人虔敬，就是施舍，祷告与禁食。这些虔敬的行为在神眼中重要吗？当然重要！因为太重要了，因此神看重的不但是做出来的行动，更是看重做这事的动机。为何神看重动机呢？因为我们做那事的动机显示我们是为何做与向谁做的。即便我们的信仰生活有很多要做的善行，有些像施舍一般的善行是为了带给人益处的，但是那本动机是因神而做，且向神而做的。但是，如果我们是做在人前的，为了得人的称赞，圣经说我们是已经得了赏赐。换句话说：我们就只得自己所要的。试想这样的人的灵性是如何的呢？他是越来越喜爱活在人前，或说越来越离开神的面。

2）做在人前的诱惑 #

我们要谨记，人最大的偶像就是得人的称赞与荣耀。因为凡事要从人得的，不会想从神得。凡事崇拜人的，不会想崇拜神。确实，我们生来都是为了得父母、老师、上司、名望之人等的称赞，因此我们所做的所有的事都是做在人面前，但是当我们重生之后，我们最首要的就是学怎么活在神面前，那也是我们一生最重要的属灵功课。因为这是所有善行的开始。因此，在施舍的事上，主甚至说不要叫左手知道右手所做的。这就是说我们在施舍的事上，甚至不感觉自己正在做什么值得表扬的事，唯有主是那善行的启发者，且是自己行善的目的。在祷告与禁食的时候，面对的都是那在暗中的父，不是要将自己的敬虔显示给人。这样一来，我们的信仰就是一个活在神面前的信仰了。我们的信心与我们的善行也是相称的。

3）在暗中察看我们的善行的父，必定报答我们 #

有些传道人解释这处经文时，马上就说神会在物质方面或是人看得到之处赏赐我们。其实，他们在解经时不严谨。首先，我们要问为何圣经说“在暗中”呢？因为那是没有人知道的。人只能看见我们外在的善行，但他们看不见我们在暗中行善的动机。同样的，神也在暗中赏赐我们。正确来说，那赏赐也是人看不见的。那是什么呢？那就是神向我们仰脸。当我们面对神而行善时，我们其实最渴望的祝福不是别的，乃是在我们的心灵里看见神用祂的脸光照我们，赐恩给我们，赐我们平安。这样的人不但在跟从服侍主时越来越有确定的心，能凡事刚强壮胆，也更加的喜爱将一切的善事行在祂面前。今天，为什么我们在行善服侍主没有能力呢？为什么这么容易胆怯呢？为什么人一有意见就使我们的心摇晃呢？因为我们常常行在人前。因此，我们对神借着祂话语给予我们的声音与指示没有定见。那么，神会不会在物质或是人看得到之处祝福我们呢？当然也有可能。但因为这是在暗中赏赐的，因此我们的心里非常的清楚那是从主得的。确实，我们常常看见有些学生考到好成绩，认为是因自己努力而来的。有的升职加薪，认为是因自己在公司里表现好而得的。但活在主面前的人晓得那都是从神赐福的恩手得着的。

  1. 最后，活在神面前绝不是脱离现实 #

1）在神安放我们的现场中活在神面前 #

有些人说：我只有活在神面前，不需管别人怎么看我，这也是一个很大的误解。倘若我们问神在哪里，客观的说祂是运行在万有里，但主观来说祂是运行在我们相关的现场里。因此，当我们说活在神面前时，我们绝不能脱离自己的现实生活去追求神的面。当我们从读经祷告的隐秘处走出来之后，我们马上是回到我们的现实生活中。在那里，有我们的个人生活、家庭、学校、工作、教会等。因此，当一个人常活在神的面前，却没有爱惜光阴，没有设想旁边配偶与儿女的需要，或是在工作上常常不尽责，处理事务敷衍塞责，还是与人相处没有智慧等，且不在这些事上多寻求神，那他在活在神面前的事上还不够踏实。

2）活在神面前的人是自然将光照在人前 #

主说：你们的光也当这样照在人前，叫他们看见你们的好行为，便将荣耀归给你们在天上的父《太5:16》。因此，我们一定要分清楚信仰的个人化与信仰的普遍化。没错，主是与我们个人同在的，且叫我们寻求祂的面，但主没有叫我们从个人走到隔离。主乃是叫我们从个人走到人前。在人前中将那看不见的主的慈爱、智慧、信实、良善等活出来，好叫世人都因我们得见神的美德，将荣耀归于祂。其实，我们特别要关注我们与人的关系。我们有时与很靠近的人一直出现摩擦，那是与我们坚持己见而不听从主有关系的。有些夫妇常常忙着工作而忽略与彼此的沟通，有的母亲常常与儿女有摩擦或是忽略他们，有的基督徒常常与同事有过节，无法合作。为什么呢？其实，那多半是我们的固执带来的。为何固执呢？那是因为我们没有活在主的面前，因此眼光、判断、决定都是以自己为中心出发。倘若我们真是一个活在神面前的人，我们不会这样做的。我们反而是会受圣灵指教，懂得快快的听，慢慢的说。我们也懂得考虑到人的处境与感受来行事。在办公中也会求上进，在人托付我们的事做得尽善尽美，不是草草了事。另外，在一般生活处事中，常常捉紧时间，将重要与次要的事物分得很清楚，且能有效的将分内的工作完成。这样一来，旁边的人对我们基督徒与我们敬拜的神就带有一种崇高的眼光了。

3）活在神面前的人绝对不会忽略大使命 #

大使命是我们主终极的所愿了。主在升天前嘱咐我们去使万民做祂的门徒《太28:19》，那也应该是所有基督徒终极的目的了。无论我们在生活处事做得怎么好，若我们是一个不传福音的信徒，我们活得只像是一个世界的好人，不是神的仆人，或说我们只是教徒，不是门徒。我们教会所有的服侍都要有一个绝对的定向，就是要传福音与培养门徒。在今年，当我们注重改革的教义，我们不能忽略传福音与牧养的事。当教义正确了，我们一定要同时求主为我们打开传道的门。要不然，我们所强调的正道就只是一种不能活化出来的道理而已。唯有当我们开始传福音，接触新人，扶助软弱的，我们才能被迫应用我们一来所学习与装备的真理。
