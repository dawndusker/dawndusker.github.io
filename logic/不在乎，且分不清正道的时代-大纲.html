《约1:1-5》《罗1:18-21》 #

序论：今天，我们或许是活在教会历史里最黑暗的时代。但我们要明白那是怎样的“黑暗”。那是因互联网所带来给全球人的影响，这世代的道德堕落，或是这个世界现在一直有很多的战争、灾殃、病毒等吗？这似乎正如很多现代教会都在提倡“末日的末时要来了”这类的信息，好叫信徒将很多的心思与关注放在现在的实事与新闻。确实，全球性的灾难是很容易占据人的关注，且将人的眼光局限在眼前所发生的事。当然，这些都是我们人看为相当黑暗的事件，且是叫人心充满不安、忧虑、惧怕的事。但是，我们要晓得圣经看为黑暗的时代并非是先从外在事件看起，乃是从内在因素看起。那内在因素就是人对正道毫不在乎的问题。

今天，我们看见全球有无数的教会。单单新加坡这小国本身有500多间教会。但我们有否问：“何谓主的教会？教会被建立的本意何在？”教会就是真道的居所，也是借着真道被设立的。《雅1:18》说：神按自己的旨意，用真道生了我们，叫我们在他所造的万物中好像初熟的果子。这就如同我们的生命是借着圣灵透过真道而生的。不是借着人意，人的方法，人的推动，人的节目而生的。当神的真道被传扬时，神的子民将自然而然地被吸引而来。教会的首要使命不是做义工，不是帮助穷人，也不是搞节目让大批外邦人进来教会感受圣诞气氛，乃是将主的真道传扬与发送出去。

因此我们要问：如果我们不在乎真理的话，为什么我们要建立教会呢？那么，如果你的讲台是有真理要传扬的话，为什么怕开教会呢？最近中国大陆政府在三月份要施行新的宗教法案，禁止任何人在网络上发布有关宗教的内容，或是用任何的社交媒体平台来传教。很多弟兄姐妹就纷纷退群，甚至很多网上查经班都要被禁止了。在这样的情况，教会应该怎么做？其实，我们首先要问的问题不是：“在宗教法令被施行，我们应该怎么办？”我们要问的是：“我们要传扬的是否是上帝的真道？我们所确信与传扬的是否符合圣经的正道？”倘若是，为什么害怕呢？难道人的手能够禁止上帝的真道吗？同样的，为什么我们教会在疫情期间不断地聚会，也必要聚会？那个心志从哪里来呢？那是因为这个教会的祭坛有使命要向这时代继续传扬真道。倘若我们是喜欢搞人数运动，喜欢热热闹闹，那么我们就应该在疫情期间关掉了。我们没有继续的理由。圣灵也不会帮助与坚强我们。

因此，我们思考问题的中心一定要放在真理，不能放在决策。决策是圣灵会引导的，但真道是圣灵嘱咐我们必要传扬的。唯有当我们认知自己必要传扬的，我们心中才会出现引导。因为若我们不是一个能为真道而舍己的人，我们的心中就会不由自主地出现很多的顾虑。我们想东想西之后，但最终还是得不着圣灵的心肠。试问现今人最关注的是什么？他们最关注的是关系与和平。因此，我们在教会天天听到的信息是：“我们与主的关系，与人的关系最重要”。没错。但“真道”是断定关系的决定因素。倘若我们与一个人的感情很融洽，我们要问的是我们中间的关系有没有真理。我们是否因明白主的道而理解彼此都是罪人，且以主的恩典与怜悯彼此相待。我们有否从主的道来学习怎么和睦共处，且向彼此活出很真实的一面？因此，若我们没有真道，我们很难讲关系。那关系也很容易留在一种正面的感觉而已。那个关系也很难达到荣神益人建国的果效。

1. 万物是借着道而造的 #

1）道创造了万有 #

约翰用“道”来形容圣子那自有永有的存在。祂是与神不同位格的，但也是与神在一起的，但也同时是神。神借着道让一切在空虚、混沌，黑暗之中的，能以进到意义、次序与光明中《创1:1-3》。神不是借着能力或能量，也不是借着爱，神乃是借着“道”来创造宇宙万有。因为这道，一切被创造，一切继续运行，一切也被维持着。在这个道里面，能力运行、慈爱显明、意义显明、关系建立、万有和谐等。圣经说：万有都是借着祂造的。然后，这道成了肉身，就是我们主耶稣基督了。

2）不认识道，等于不认识耶稣 #

今天，很多信徒说：我们不需要认识道，我们认识耶稣就好。试问那是哪里来的耶稣呢？那是谁介绍的耶稣呢？其实，那是圣经所记载的耶稣。因此，若我们不回到圣经认识神的道，那等于我们不认识耶稣，那也等于我们不认识耶稣里面的一切属性。我们若谈到爱时，我们多半讲的是人世间的爱，是人的博爱。我们讲到智慧，那多半是地上的智慧。我们讲到公义，那是人片面的公义。我们讲到能力，那多半都是那种人感觉到的能量，能带来奇特现象的能力。我们不认识运行在万有的能力，且重生与改变人生命的能力。这一切都是我们因不认识道而产生的结果。

3）一个基督徒所得的呼召应该是从他所听的福音之道而来 #

今天，很多基督徒的呼召像是来自一种为事工的激情。有的说：我对年轻人有负担。有的说：我对圣乐有负担。有的说：我对探访有负担。当然，我们被呼召在不同的岗位上服侍主，但我们要问什么促使我们对这些事工有负担？倘若我们只是要帮一些年轻人，像是给他们积极向上的影响，那这世界的辅导家也能做到。或者，若我们只是要作出优美的音乐来让会众来唱，那意义也不大。其实，我们每个服侍的潜在动机都是要追求、维护、传扬真道。例如：教会的辅导工作是要将人带向真理，且用真理来教导、劝诫、安慰，甚至是归正他一些错误的思想，且叫他顺服神，不是单单给他感到舒缓；或者教会的圣乐也不是单单找好听的诗歌来献唱，乃是借着诗歌来表明我们的信仰，且让会众唱出自己所信的正道。因此，真正好的诗歌不但是一直强调这神如何喜爱我、帮助我、满足我，乃是让人认识这位神和祂的属性，好叫我们宣扬祂，且将颂赞归给祂。

2. 这时代在认识正道的事上有三方面的危机 #

《罗1:18》清楚告诉我们：人最严重的罪不是别的，乃是行不义阻挡（压制）真理。因此，我们的罪是内心里不愿意追求，甚至是否认神向我们显明的启示。人不接受真理，因此不接受基督。因此，当一个人宣称他接受基督，但不接受基督里头的真理，那不是真接受基督的。这样的人的救赎是可质疑的，他也没有办法走成圣的道路。

1）不在乎是否有真假之道 #

可能这时代最黑暗之处是人一般上都不在乎到底有没有绝对的真理。大部分人都认为只要行得通就行。他们真正相信的是【实用主义】，并非客观的真理。倘若走进教会，或是在教会里服侍，甚至是做教导工作的人都是这样，教会一定会走到一败涂地的光景。因为对“何谓真理”的一个基本关心都没有的话，教会根本就是一个俱乐部了。在这样的教会里，只要任何的道理或形式能够叫人感到舒畅、愉快、自由、能应付生活，那大致上都能被接受。因此，我们看见有些牧师宣称说常守圣餐就能从身体的病痛得医治，许多教徒就天天在家中自己守圣餐。若说到用油来涂抹头就能得恩膏的话，许多教徒就买很多橄榄油给自己涂油，不但自己涂，也给旁边的人涂。切实的说，这等人不在乎真假之道，因为只要能讨自己耳目喜悦的道，或是实用的，他都能接受。他们如同《提后4:3-4》说的那种人。他们其实厌烦纯正的道理，耳朵发痒，就随从自己的情欲，增添好些师傅，并且掩耳不听真道，偏向荒渺的言语。这等人的心田已经坏了，根本是不能结果的。对于这样的人，唯有很严重的问题临到他们，他们才会问：何谓真道？何谓终极的真理？但可能跟他们解释之后，他们稍微有所谦卑，但只要问题一结束了，他们还是回到自己原有的实用主义概念。

2）真假之道分不清 #

《提后3:7》说：这等人是常常学习，终久不能明白真道。真正的道是什么，他们始终分不清。为何如此呢？理由很多。有的可能是被很强的私欲牵引。他们爱肉体与世界的心叫他们在领受神的话语时一直很模糊。有的可能曾经很深地进入一些人的宗教或错谬的教导，因此不能明白圣经所讲的基要真理。例如：他们一直重视人的功德，以致不能明白神的恩典。或者他们一直在乎外在的形式与仪式，却不能用心灵的新样来敬拜服侍主《罗7:6》。有的是一直留在一些错误教导的教会里，每周听的都是错谬的道，因此无法归正。这等人的心灵里有很多的干扰。圣经告诉我们这等人在真假之道分不清时，一直处在一种模糊与无知的捆绑里，因此他们的行事为人不能荣神益人。这样的人一定要经过一个“全部拆毁，重新建造”的过程，借着圣经的话被教训、督责、归正、教导学义《提后3:16》，整个的身心灵、生活、人际关系、经济管理、服侍等才能得医治。

3）即使认识真道也不坚定 #

这如同罗得身上的问题。今天有好些信主多年的信徒的确多年在教会领受真理，但是一直放不下各样的重担，脱不去容易缠累他们的罪。他们无法忍耐，无法坚定奔那摆在自己前头的路程《来12:1》。结果，他们的一生就是做一个妥协的基督徒。换句话，他们所认识的真理没有办法坚强他们的意志，使他们坚定不移地跟从主。当然，他们因为良心受谴责，所以常常会处在唉哼的光景。他们可能会说这时代的试探非常多（例如：互联网、工作忙碌、资讯泛滥（google）等），但原因只有一个，就是他们不愿意从神的真道来寻求能力。他们学习神的道不是系统性与全面性地学习，乃是以实用性的角度来学习。因此他们的思想没有被归正，反倒是存有模糊。要不然就是没有借着确认、察验、实践来活用所认识的真理。因此，这样的人常常就是处在一种无可奈何或是被罪疚感捆绑的光景。

3. 在这危险的时代，主呼吁我们回归与持守正道 #

1）我们主耶稣是真理的化身 #

我们主耶稣基督不是我们人造出来或想象出来的救主。祂是真理的化身。祂说：“我就是道路、真理、生命。若不借着我，没有人能到父那里去”《约14:6》。因此，耶稣对那些敌挡祂的法利赛人说：“你们是出于你们的父魔鬼，你们父的私欲你们偏要行。他从起初是杀人的，不守真理，因他心里没有真理。他说谎是出于自己，因他本来是说谎的，也是说谎之人的父”《约8:44》。这就说明这些人敌挡基督，是因为他们本是敌挡真理的。他们如他们的父，心里没有真理。

2）这世界充满虚假 #

这世界的问题不是战争、疾病、病毒等的问题，乃是在这些灾殃与灾难中，仍然没有转向真理。他们在虚谎里面活着，设法应对问题，甚至是制造出更多人的宗教来安抚自己的良心，要不然就是依靠这世界的伟人。即便他们得着了很多神的恩典（各样美善的恩赐与全备的赏赐都是从神而来），他们不因此荣耀神，也不感谢神《罗1:21》。人的问题都是活在这空中掌权者的首领之下，被他的谎言蒙蔽，因此国攻打国、民攻打民、家攻打家、夫妇攻打彼此、父子攻打彼此等。因此，我们每个基督徒的使命就是要将人带到神的真理面前。若我们在这样的时代不在乎何谓正道，且只有将神的话语当作一种使用工具来给予自己暂时的解脱，那我们根本对这时代是毫无用处的。我们就必如早晨的云雾，又如速散的甘露，像场上的糠秕被狂风吹去，又像烟气腾于窗外《何13:3》。我们所做所说的根本没有喂养他人的灵魂，我们工作的果效也无法影响下一代。

3）信徒一生的成圣都是借着认识真理而来 #

在最后晚餐，主耶稣为着门徒向父祷告说：父啊！他们不属世界，正如我不属世界一样。求你用真理使他们成圣，你的道就是真理《约17:17》。试问我们能用自己的意志或决心来成圣吗？在新的一年，我们能用自己的决断来改变自己的前景吗？其实，若我们没有回归正道，且坚定持守自己已经明白的，我们在世界就会活得像一个自相矛盾的人，甚至是过着一个苟且偷生的人生。确实，成圣的路是要靠着真理的圣灵才能成就的事。圣灵用真理光照我们的心思意念，使我们心意更新而变化，我们才能在凡事上察验何谓祂善良纯全，可喜悦的旨意《罗12:2》。真正的基督徒不会对正道马虎，因为他认识圣经的目的不是给予自己解脱或满足自己的需要，他乃是要认识这位神和祂在自己身上的作为，好叫自己能行在祂的旨意中。因此，在我们教会继续这样成长的过程，有些弟兄姐妹可能带着自己的问题来到当中，他们最需要的是回归正道，清楚知道神的道是将我们指向神，且从认识神来面对自己现处于的问题。对另外有些已经归回正道的，今年的祷告要多放在持守正道，且在实际的生活现场中将此应用出来。

不在乎，且分不清正道的时代-讲章
