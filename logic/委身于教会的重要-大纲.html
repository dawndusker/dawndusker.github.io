委身于教会的重要 #

《弗3:16-19》《帖前5:12-15》

序论：当我们回归正道之后，我们进入了一个“活在神面前”的信仰生活。但是“活在神面前”仍旧是我们个人与神之间的事。当我们个人陷入软弱时，我们自己怎么帮助自己呢？当我们要行善时，我们向谁行善呢？因此圣经没有将我们个人与神的关系视为基督徒信仰生活的结论。圣经反倒是将我们集体与神的关系视为信仰生活的结论。在《弗3:17-18》，保罗不但愿意基督因我们的信住在我们心里，叫我们的爱心有根有基，他也愿意我们能以和众圣徒一同明白基督的爱是何等长阔高深。 《来3:13》也嘱咐信徒总要趁着还有今日，天天彼此相劝，免得我们中间有人被罪迷惑，心里就刚硬了。因此，对一个真正的基督徒，当他活在神前的时候，他同时会将自己放在一个能勉励，以及监督他的群体当中，那群体也就是主的教会了。今天，我想将我个人在教会长大的过程说一遍。首先，我因着能自小在教会长大而感恩。虽然我不是生在一个基督徒的家庭，但因着神的恩典，母亲从小就让我到教会去，因为觉得人去教会起码不会学坏。因此，教会所给予我的基督教文化就从小深印在我心中。对我而言，彼此相爱，彼此饶恕，彼此关心，彼此服侍并非是陌生的字眼。这些都是从小在教会生活中实践的。再就是，什么是圣经的权威？什么是崇拜的庄严？什么是圣诗？什么是奉献？什么是教会纪律？什么是基督徒的义务等？这一切和其中的精神对我而言都不陌生。因为这一切都在教会生活的文化中看得到。有时候，当我们听一个字眼，我们并非会晓得那意思，直到我们渗入其中的生活。我自己在教会长大的过程中明白什么是圣徒相通。虽然曾经年轻时有年轻人的喜好，也有年轻人喜欢的去处。但我的中心点一直以来都是教会。可能我在长大的过程也有在外面听流行歌，或是与世俗的朋友寻欢作乐，但我都因为教会生活被约束，甚至也很多时候即便爱世界，但因为与众圣徒的关系被挽回，以致没有成为迷失的羊。渐渐地，我也成熟起来，灵命也开始稳健，开始明白聚会的价值、服侍的价值，且将奉献带给教会的意义。这一切都是我在主面前忠心做的事，且确实认定施比受更为有福。到了一段时间，我发现人生里很重要的祝福都是借着教会生活得着的。我在教会认识我的妻子，也在教会生养我三个儿女。可能在这当中，我不像很多的父母能在每周末带孩子出门玩乐，我们家庭的娱乐时光多半是在教会度过。这当中，我的孩子认识不同的叔叔阿姨，也认识其他的孩子。我和我的家庭都明白这些每周见到的人就是我们的家人，是与我们永远会在一起的肢体。即便与这群肢体一同生活服侍中也经过很多的摩擦，但渐渐地，我们适应彼此，也学习怎么在主里切实地相爱。这当中，我们也时不时成为了警戒彼此与担代彼此软弱的战友。总而来说，主给予我的这群肢体不但是我祝福的对象，也是神用来祝福我的管道。今天，我们看见教会生活进入另一种文化。过去那比较严谨与端正的文化已经被那自由与松懈的文化取代了。这种文化表面看像是给人很多的空间。它根本不需要参加者的委身，只求人能进来教会做个礼拜，然后将一些钱捐给教会。这样的教会也不讲教义，也不讲基督徒义务，更不讲什么教会纪律。因为怕这些会将人从教会赶走。今天很多这样的信徒就开始认为在家里做崇拜也是合理的，他们不需专属于一间教会。对于教会，他们想去就去，不去也罢。有些则是到处参加特会，专追寻那些新鲜的事。他们觉得这才是给人自由的基督教信仰。因此当疫情一发生，他们连挣扎都不挣扎，就自动留在家里看直播。渐渐地，主日就成为了他们家庭用来出外娱乐的放假日。他们心里无愧，也不会想念与肢体的关系。这就是当代许多灵恩与福音派教会已经陷入的光景。确实的说，这已经不是神给予基督徒的信仰生活。这乃是一种新潮的基督教，是一种教会界里的实用主义信仰。除非我们看见这危机而懊悔，这种不需委身于教会的风潮将会横扫这个时代，许多教会最后就会沦落到老弱病残的属灵光景。

1. 信徒不委身于教会的光景将显示在几方面 #

1）信徒以消费者的心态来领受福音 #

在现今教会里，我们看见“基督教媒体”开始胜于“基督教教会”。从互联网的盛行，基督教信息开始遍满社交媒体。从那时起，信徒开始能够上网寻索信息聆听。这也带给现今教会很大的挑战。因为从那时起，信徒们能够很方便地从网络上领受信息，且随意挑选自己喜爱听的道，且随意地将这些道转发出去。从那时起，教会不再能够守护信徒的“耳朵”，信徒也不愿意以教会信息为信仰的导向，因此很多的信徒就这样被错谬的道误导，常常学习，终久不能明白真道《提后3:7》。当然，这样的事产生能从很多角度来反思。其中一个原因也是因为教会的讲台变弱。现今传道人没有效法使徒的榜样，专心以祷告、传道为事《徒6:4》，反倒专喜爱管很多教会琐碎与繁杂的事物。因此，羊群便在外寻草吃，结果有好些落入错谬的假道。整体来看这是一个恶性循环。但从这里，我们要问神为何许可这事？一方面，神要那些忽略讲台信息的传道人悔改，因为他们没有担起牧养群养的责任，因此自食其果。但另外，我们也要问一个很基本的问题，就是当一个信徒在外领受了灵粮，是不是应该与那治理与劝诫他的牧者们求印证才将信息传开？答案是当然的！因为牧者是为全群谨慎的监督。当群羊这么做，就是尊重神设定的权柄，自己也受那权柄保护。

2）信徒不愿意受到治理与劝诫 #

当信徒在领受神的话语上成为消费者，他很自然的倾向就是不愿意受到治理与劝诫《帖前5:12》。试想若一个人喜欢听什么，那就是他的“福音”，那他又怎么愿意附属在治理与劝诫他的监督之下呢？像这样的人，他说自己信了而认真的会考虑受洗吗？他不来聚会时会请假吗？他会愿意向弟兄姐妹交代他目前的情况吗？他会愿意多参与服侍吗？他会愿意将自己当纳的十分之一献上，与教会一起同得基业吗？倘若一个信徒没有基本的教会观，清楚知道圣经命令信徒当附属在教会领导权的治理与劝诫之下，信徒都会按自己的喜悦选择做或者不做这一切。其实，这是一个在人里头随心所欲，不受约束与管制的性情。有些信徒可能会说，“我们受神的管制，不受人的管制”。那试问神的管制是只有在我们内心吗？难道不更是透过看得见的教会牧者与监督吗？就是连监督与牧者也要顺服会众，在众人有看法之处不能一意孤行。因为圣经说当存敬畏基督的心，彼此顺服《弗5:21》。

3）信徒之间失去公义与慈爱 #

《帖前5:15》很清楚说不可以恶报恶，待彼此要追求良善。试想肢体之间怎么以恶报恶呢？确实，我们看见有些教徒在背后彼此中伤彼此，这是一间教会里非常要不得的事情。久而久之，人的心就会生出毒根。其实，如果一间教会已经缺乏了那基本的纪律，每一个人来到教会的心态都是会将自己举高的。这种心态叫人来了教会，只渴望自己被伺候、被善待，不是要来教会服侍人、善待人。倘若牧师的讲道不合他胃口，他就缺席。倘若教会在用钱方面不合他意思，他就不奉献。倘若招待员不将他要的位子给他，他马上抱怨。其实，这种举动都是源自一种高举自己的心态。当我们如此行，有否问过神的公义与慈爱在哪里呢？但试问信徒为何都这样的随心所欲呢？因为大家都已经习惯性的不受到教会任何的约束了。但这样的人也就越来越得不着属灵的保护。他们的信心容易软弱，心里也容易生出毒根。久而久之，我们就会看见神的管教临到他们。

4）越来越多信徒走向一种“无教会主义”的光景 #

因此，我们会发现有一种时代性的“无教会主义”出现，就是宣称自己是属于主宇宙性的教会，但却不愿意附属在任何的地方教会之下。要不然，就是按着自己的方便与喜悦常常到不同的教会聚会。当他这样做，他从不需要向任何的信徒群体交代。他不需要用神给予他的恩赐来服侍教会，也不需要奉献给主的教会，也更不想让人干涉他的生活。其实，这样的信仰对自己是最不健康的，因为当他不专属于一间教会，他根本没有办法行善。因为基督徒的善行都是向着旁边的肢体做的。他也没有办法受到任何群众的代祷与保护。这样一来，他就像一个离开羊圈的羊。因此，当一间教会渐渐大起来，一定要定出一些属于教会的规矩。例如：成为教会的会友应该具备哪些条件？教会的监督（牧师，长老，传道）也应当腾出一段慎重的时间来与这些会友见面交流，表示欢迎他们加入教会。

2. 为何要委身于当地教会？ #

1）因为愿意要顺服神的话 #

愿意要顺服的信徒首先知道圣经的书信都是写给教会的。因此，他不可能说相信圣经的话却不属任何一间教会。另外，一个愿意顺服的信徒知道自己的软弱，因此他一定愿意要向周围的信徒负责。因为当他委身于教会，他也晓得自己所参与的聚会与服侍都会直接或间接的影响到周围的肢体，因此他会被约束不会随意停止聚会，或是在自己的服侍松懈。最后，他晓得圣经里所有的德行是先向着主内一家行，才向众人行。《彼后1:7》说：有了爱弟兄的心，又要加上爱众人的心。《加6:10》也劝信徒说：有了机会就当向众人行善，向信徒一家的人更当这样。因此，愿意顺服神的话语的信徒一定不会让自己沦落到无教会的光景。

2）因为相信圣徒相通 #

若没有圣徒相通，“顺服”只是一个概念。圣经里充满了“彼此”这个字。例如：彼此相爱、彼此饶恕、彼此体恤、彼此劝慰、彼此建立德行等。《来10:24》告诉我们：要彼此相顾，激发爱心，勉励行善。确实，圣经里没有一个善工是独自一人做的。我们不是与旁边的肢体一起同工来做，就是做在另一个肢体身上，像是做在主身上。《彼前4:7-8》说：“万物的结局近了...最要紧的是彼此切实相爱，因为爱能遮掩许多的罪。”今天，教会界有很多的教师与信徒在谈论末世论。但很奇怪的是他们并没有投入教会生活。试想若我们深信末世的末时真是即将来临，我们不是最想与众圣徒在一起吗？我们不是快快会忘记彼此的过错吗？因此，我们可以说不看重圣徒相通的信徒并不是真信主快要再临的人。他们只不过是被末世论谈的好奇感吸引而已。

3）因为愿意活在权柄之下 #

在现今世代，很多人不喜欢“权柄”这个字。他们认为这是一个人用来控制他人的手段。特别，当基督教经过罗马天主教带领的黑暗历史后，圣徒被教权压迫，很多信徒因此感觉教会的权柄不是一个很正面的字眼。特别，在现今这么自由的社会里，很少人会乐意将自己附属在另外一个权柄之下。但《帖前5:12-13》说：“弟兄们，我们劝你们敬重那在你们中间劳苦的人，就是在主里面治理你们、劝诫你们的。又因他们所做的工，用爱心格外尊重他们”。圣经在此嘱咐信徒当敬重教导与带领他们的教会监督。特别，若监督在他们身上没有权柄，信徒们怎么会听从他口中所教导的神的话语呢？这样一来，信徒们又怎样能受到神话语的保护呢？因此保罗告诉提多当以权柄带领教会。他说：这些事你要讲明，劝诫人，用各等权柄责备人，不可叫人轻看你《多2:15》。

4）因为重视基督徒的公开身份 #

《弗4:1》... 既然蒙召，行事为人就当与蒙召的恩相称。当我们做一个委身于教会的信徒，我们周围的人很快察觉到我们的生活是分别为圣的。可能我们周末要参加主日聚会，周间要参加小组与门训。久而久之，我们在人前就显露出我们是一个敬虔的基督徒了。试问我们是一个愿意公开自己基督徒身份的信徒吗？还是我们宁可暗暗的做一个基督徒呢？主耶稣说：凡在人面前认我的，我在我天上的父面前也必认他《太10:32》。从另一方面来说，公开承认自己信仰的基督徒也是一位有可能向他人传福音的信徒。

5）因为愿意众教会都晓得自己所领受的真理 #

今天，若你是一位委身于某某教会的信徒，其他教会听了你教会的名字，也会略略知道你所信的教导是什么。倘若我们是属于新加坡基督生命堂，人一上网寻索就会知道我们所传的是什么教义。新加坡基督生命堂可以说是一间改革的独立教会。我们的神学理念是以改革教义为基础。我们所呈现出来的是以马内利（神与我们同在）的生命。因此，我们可以说是一间【改革教义为基础，以马内利为生命】的教会。逐渐的，当人一听到“新加坡基督生命堂”这地方教会，就会知道这里头的成员所听所信的是什么。因此，我们不但向外邦人展现我们公开的基督徒身份，也向众教会的主内肢体陈明我们的教义与信念。这样一来，我们才能堂堂正正的成为众教会的一份子，不是那些传异教或是偏颇之道的教会。他们好像是挂羊头，卖狗肉一般。他们从不将自己教会的本信念陈明，只有以实用主义来勾引人进入他们的教会，之后却将他们带去信从另外一个福音。
