信三位一体的神 #

创造宇宙万有和规律，掌管世界，天使，人类，全知全能。圣洁，公义，慈爱。

亚当 #

人起初有从神来美好的形象样式，与神同住在伊甸园，被赐权柄管理世界。为要荣耀神。

人的罪 #

因亚当犯罪，与神隔绝，在属灵上死亡，从亚当而出的后嗣，与神的关系断绝，无法行神要求的义，止不住犯罪得罪神，亏缺了神的荣耀。

罪人的处境 #

罪人逃不出全能全知神的监察，神必要以圣洁，以公义审判人，神以将来永不熄灭的愤怒设立地狱，以永火刑罚罪恶。

神的拯救 #

在造人起初，人早已知道人会犯罪，为人预备了救恩，在旧约的历史不断预言启示神的拯救。在末世，差遣爱子降卑为人，承受他百姓当受刑罚和审判，并将自己连于天父的生命与人共享，预备新天新地给信他之人，在永恒中合一。

耶稣的福音 #

无罪的耶稣钉在十字架上，担当了信他之人的罪，承受了神的愤怒，亲自做了赎罪祭挽回祭。最后死里复活显明是神的儿子，无罪而替罪的羔羊。

圣灵的重生 #

圣灵开启，使人明白基督的十架代救赎的大爱，醒悟自己的罪，使人以信心回应圣灵而重生，住在人心里，引导人明白圣经，真理，保守所信之人不再失丧。

我的信 #

明白自己的罪和神为罪人脱离死和罪的福音，为罪悔改，愿意信靠基督，求得赦免和保护，并立志要脱离罪，顺服基督，舍己跟从。努力过与福音相称的生活，警醒等候主的再来。

学习圣经 #

圣经是神的默示，神向人关乎生命和敬虔的启示也只在圣经，是信徒和教会最高的标准，基督徒要在正意理解圣经的基础上成长，学习得救成圣的智慧。
