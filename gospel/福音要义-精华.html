信三位一体的神 #

创造并且掌管宇宙万有，包括一切的秩序，规律，天使，人类。全知全能。圣洁，公义，慈爱。三位一体的神同享神一切的属性，在永恒中同尊同荣，以完全的爱彼此相属，所以神就是爱的本体。而圣灵顺服基督，基督顺服天父，又构成位格中的完全顺服的关系。当然我们对神的理解都来自圣经的启示，神只是向我们启示了我们所能知的部分。

亚当 #

人起初有从神来美好的形象样式，有从神而来的思想，理性，情感，智慧，创造性，自由意志。 也有圣洁，公义，良善，爱的道德属性。与神同住在伊甸园，被赐权柄管理世界。为要显明神的形象样式的荣美，以此荣耀神。起初，无罪而美好的人和圣洁的神和睦相交。

人的罪 #

因亚当犯罪，与神隔绝，在属灵上死亡，从亚当而出的后嗣，与神的关系断绝，从此罪，死亡，魔鬼的权势统治了人类。 人和神关系破裂，人在恩典中堕落，人和人，人和自我，人和自然，人和灵界的关系都败坏了。 从亚当而出的人都失去了和神正当的关系，被神拒绝。 也自然的不认识神，拒绝神，思想，形式与神反对。根本无法行神要求的义，止不住犯罪得罪神，亏缺了神的荣耀。 人开始活在满足欲望，建立骄傲的动力中。 无视神本身，也包括神的荣耀，权柄，尊贵，圣洁，良善。神所当配得承认，尊崇的荣耀，被人无视。 人在谋取自己的荣耀，满足。 积极的自我成全。 人无法自发的从罪恶中脱离，止息不住犯罪的冲动，无论怎样加工，美化，调整。都不过是在行犯罪之实。 人的义都是污秽的衣服。

罪人的处境 #

圣洁公义的神，鉴查罪恶，以罪恶为罪恶，必要审判罪恶，对犯罪的人，毫不姑息。 神绝不会忘记，略过罪人的罪，所以将要临到人的是神永不止息，和纯一不杂的忿怒。 人只不过活在一个神暂时的宽容，忍耐中。 积累罪的工价，积累神的愤怒，直至末后的审判。

罪人逃不出全能全知神的监察，神必要以圣洁，以公义审判人，神以将来永不止息的愤怒向罪人倾倒，以地狱的永火刑罚不能悔改归信的罪人。

神的拯救 #

但神有怜悯的心肠，绝不愿意人落入地狱中。 并且神本意要创造的人，就是按照他的形象和样式。 并且万有的是在基督里造的，通过基督造的。 神要通过基督，在基督里，进行新的创造。 真正完成神造人的目的。 神的旨意必不落空，神的慈爱和良善，在造人之初就向着人类展开。 神乐意将他的丰盛，荣美，尊贵与人分享。 愿意与人交往，使人领受他的丰富，大能，智慧。 以至于人把神荣耀的本质，看为最本质，最高的荣耀。 使人荣耀他。 但罪恶中的人，根本不可能达成神的目的。 他口发不出赞美，感谢，反而是亵渎，辱骂。 他的心发不出尊崇，敬拜，只有无视，拒绝。

所以神还有行动，就是在创立世界之先，就预备救赎。在人类的历史中行动， 与亚伯拉罕立约，预言，预表的形式表明救赎事件。 到了定的日子，基督由圣灵感孕童女马利亚，神降生为人。 穿上人的软弱，局限，身体性的需要，生活种种试探。 只是耶稣没有犯罪，完全以天父的旨意为使命。传天国的福音，征服黑暗，光照人心，抢夺灵魂，建立教会，训练门徒，托付使命。 而耶稣明明知道必然的，极致痛苦的死亡，却顺服到底，欣然赴死。无罪的耶稣却成为替罪的羔羊，以人的生命，在肉身中定了罪案。 经历了神的离弃，审判，击打，死亡。 原来耶稣所受的苦，承载的却是罪人的罪。神对罪人的愤怒，加在了耶稣身上。 后又从死里复活，真正胜过了罪，死亡，魔鬼， 也成为门徒可信的凭据。 同事基督也以我们看不见的形式，作为大祭司，将自己一次献上天上的至圣所，真正在父神面前成就了救赎工作，从此成为我们的中保，为我们在父神右边代求。又为我们差遣保恵师圣灵，确保救恩的果效。 如今基督做王升入天堂，为我们预备地方。将来必带着圣城新耶路撒冷降临，将全地更新。 行善的复活的生，作恶的复活定罪。

耶稣的福音 #

无罪的耶稣钉在十字架上，担当了信他之人的罪，承受了神的愤怒，做成了赎罪祭挽回祭。最后死里复活显明是神的儿子，无罪而替罪的羔羊。耶稣也进入天上的至圣所，一次献祭，就完成永远赎罪的事，又为我们做有效的代求。 使我们也能进入幔内，使我们能被天父接纳，在至圣所与神亲密相交。 基督又差遣圣灵做我们的保惠师，引导人走永生的路。 基督也藉着教会统治管理他的百姓，为要成全弥赛亚国度。

圣灵的工作 #

圣灵开启，是神主动向人行动，使人明白基督的十字架替代性救赎的大爱，为罪为义为审判，自己责备自己。在对罪无可指望的绝境中投靠唯一的救赎耶稣基督，得以使人以信心回应圣灵而重生，住在人心里，引导人明白圣经，真理，保守所信之人不再失丧。人也是在圣灵的开启中， 领受神的尊贵，荣耀，能力，爱。 以至于人能向着神感恩，爱慕，顺服，跟随，舍己，侍奉。我们也是藉着圣灵与神相交，祷告，建立活泼的关系，在人神关系的基础上去构建合神心意的人与人，人与物，人与邪灵的关系。爱神，爱人，抵挡魔鬼。 同时圣灵也给我们诸般的恩赐，恩典，使我们用来造就生命，治理神的家。

我的信 #

明白自己的罪和神为罪人脱离死和罪的福音，为罪悔改，愿意信靠基督，求得赦免和保护，实际的罪被清算，也经历圣灵的更新，赦罪之恩，救恩之乐。 不仅知道神我们的创造主，也是我们的救主，也是掌管我生命的主。通过圣灵，与基督联合，发生了我生命的改变，基督的性情在我里面，理所当然 要努力脱离罪，顺服基督，舍己跟从。努力过与福音相称的生活，警醒等候主的再来。

学习圣经：圣经是神的默示，神向人全备的启示也唯独圣经，是信徒和教会最高的标准，基督徒要在正意理解圣经的基础上成长，学习得救成圣的智慧。当然唯独神藉着圣灵的光照能使我们明白圣经的精意。圣灵藉着圣经对每一个人说话，也透过每一个人说话，帮助每一个人省察。我们在读经的过程中，被揭发，被教训，被说服，被督责，被照亮，被引导，被归正，激发爱心，勉励行善。

  1. 我们尊重先辈们对圣经的领受，神给历代圣徒的启示，参考释经材料和神学知识。
  2. 我们渴望圣灵活泼的工作，不懈追寻，尽心默想，彼此交通。使我们藉着圣经神的话语住在基督里。
  3. 我们坚持以经解经，以整本圣经解释整本圣经。因为圣经无一缺少，无一没有伴偶。
