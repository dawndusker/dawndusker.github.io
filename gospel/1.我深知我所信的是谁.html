1、普遍启示 #

康德：有两种东西，我对它们的思考越是深沉和持久，它们在我心灵中唤起的惊奇和敬畏就会日新月异，不断增长，这就是我头上的星空和心中的道德律。

内在启示 #

人的道德律，以及理性，思想，情感等内在的属性。表明创造我们的神，有无比的智慧。公义，慈爱，良善，诚实，表明着创造我的是一位充满道德属性的神。

【创1:26】　神说：“我们要照着我们的形像，按着我们的样式造人。”
【创1:31】　神看着一切所造的都甚好。

外在启示 #

世界运作的机制，秩序，规律，能量，智慧，美学。

一个华丽的建筑，必然有个建筑师；一个精密的手机，必然工程师；一个美丽的花园，必然有个园艺师；那么充满魅力，生机的世界，是不是有个智慧的创造者。
铁路的有秩序的运转，涉及到车辆，铁路，运营系统的创造和经营，以及时间间隔，车次，车速，变轨，电力供应等等，必然有一套高超的运作机制。那这个有秩序的宇宙，生命，是不是有个大能的管理者呢？

二、 如何认识神 #

1、不知者无罪？ #

【何4:6】我的民因无知识而灭亡。你弃掉知识，我也必弃掉你，使你不再给我作祭司。

【罗10:2】我可以证明他们向　神有热心，但不是按着真知识。
【罗10:3】因为不知道　神的义，想要立自己的义，就不服　神的义了。

如果真的有一位神，我们却不认识他，是不是一件可怕的事情。不懂法，违法就能避免被审判吗？那法律其实就没有约束力了。因为我们在无知中就冒犯了神。 当人对神的认知是错误的，不是真知识，那么我们也会得罪神，我们要立自己的义，比如道德感，宗教观念，理论。就不服神的义，不服神的律法，神的圣洁，神的要求，神的作为，神的审判。

你有什么资格审判我？ 你凭什么要让我听你的？ 凭什么你就是真理？

2、特殊启示-圣经 #

我们有可能认识神吗？神能不能做到让人认识他？神通过什么方式使我们认识他？ 圣经

【彼后1:21】……从来没有出于人意的，乃是人被圣灵感动，说出　神的话来。

40多位作者，一千五百多年，一个主题-救赎，一个主人公-耶稣基督 世界的创造，到世界的终结，并且有个全新的，完美的世界重新神被创造。

神是超验的，不在人的感知内，不可能像做实验那样能被人检验甚至是反复检验。真正的神，只能是他主动向人启示自己，不可能是通过人的宗教发展，把神的形象不断衍生出来，并且前后不一致，版本多元化。人不可能通过经验，见识，感觉，推理，玄想出真正的神，所以我们对神的认识唯独通过学习圣经客观的启示，而不是建立在想当然的感觉和玄想上。

3、耶稣基督 #

旧约启示将要来的基督，新约记录应验旧约预言的基督，包括基督的死和复活，并且预言将要再来的基督。而他是我们的中保，我们唯独藉着耶稣基督认识神。不仅仅是真理的知识，也包括和耶稣基督建立交往关系。

认识首富，首长，君王。在知识上是一份荣耀，在关系上的认识，是更高的荣耀，何况是我们被对方认识。我们能认识神，又被神认识，远超这种荣耀。

三、神是怎样的 #

三位一体的神 耶稣基督，荣耀的父，赐人智慧和启示的灵 圣父、圣子、圣灵

同尊同荣，全然合一，彼此相爱 同属一位神，但又是三个位格。有独立的自我，内涵，情感，意志，理性。 我们只有在关系中更清晰的认识神的位格。
如果神只有自我，就不会和其他客体发生关系。没有其他的存在能配的一位一元的神完全的爱。当然也就没有忘我的舍己顺服。
